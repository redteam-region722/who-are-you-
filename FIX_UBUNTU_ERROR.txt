Ubuntu VPS Error Fix - pyOpenSSL Compatibility Issue
=====================================================

ERROR:
AttributeError: module 'lib' has no attribute 'X509_V_FLAG_NOTIFY_POLICY'

CAUSE:
Incompatibility between eventlet and older pyOpenSSL version on Ubuntu

SOLUTIONS:

Option 1: Upgrade pyOpenSSL (Recommended)
------------------------------------------
Run these commands on your Ubuntu VPS:

pip3 install --upgrade pyOpenSSL
pip3 install --upgrade cryptography

Then restart the server:
python3 server.py --web --web-port 5000


Option 2: Use system Python packages
-------------------------------------
If Option 1 doesn't work, uninstall and use system packages:

pip3 uninstall pyOpenSSL cryptography
sudo apt update
sudo apt install python3-openssl python3-cryptography

Then restart the server:
python3 server.py --web --web-port 5000


Option 3: Switch to threading mode (Alternative)
-------------------------------------------------
If the above don't work, modify web_server.py to use threading instead of eventlet.

Change line 30 in web_server.py from:
socketio = SocketIO(app, cors_allowed_origins="*", async_mode='eventlet')

To:
socketio = SocketIO(app, cors_allowed_origins="*", async_mode='threading')

Then install threading support:
pip3 install simple-websocket

And restart the server.


RECOMMENDED APPROACH:
Try Option 1 first. It should resolve the issue in most cases.
